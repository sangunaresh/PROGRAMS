simple omlette receipe

Ingredients

  • 2-3 large eggs

  • salt, to taste

  • pepper,to taste

  • 1 table spoon of butter or oil

  • optional fillings:cheese,diced vegetables,cooked meat,herbs

Instructions

  1. Beat the eggs

  2. Heat the pan

  3. Cook the omlette

  4. Add fillings(optional)

  5. Fold and serve

  6. Enjoy
